# 甜守居家_特約契約與工作規章內控專案

## 專案定位

本專案為公司層級內控專案，管理特約契約、工作規章、違規記點、教育訓練、懲處與改善追蹤。

居督工作手冊只引用本專案，不納入完整法規內容。居督工作手冊處理「日常怎麼做」；本專案處理「哪些行為不能踩線、違反會怎樣、公司如何教育與管理」。

## 資料夾使用方式

| 資料夾 | 用途 |
|---|---|
| 01_特約契約與法規 | 放衛生局特約契約、違規記點整理、相關法規與契約附件 |
| 02_工作規章 | 放公司工作規章、居服員服務規範、獎懲辦法、工作倫理 |
| 03_交叉比對表 | 放契約條文 vs 工作規章 vs 教育訓練主題的對照表 |
| 04_教育訓練教材 | 放居服員、居督、業務負責人訓練教材、簽到表、測驗題 |
| 05_日常溝通話術 | 放居督提醒居服員用語、LINE公告、個別輔導說明稿 |
| 06_違規事件處理紀錄 | 放違規提醒、輔導、懲處、申訴、通報及事件處理紀錄 |
| 07_改善追蹤與稽核 | 放缺失改善追蹤、內部稽核、自評與改善結案資料 |

## 核心管理原則

1. 特約契約是公司對主管機關的責任基準。
2. 工作規章是公司內部管理與教育訓練依據。
3. 居服員教育訓練需使用白話版本，不直接用法條宣導。
4. 每一項違規風險都應對應到：規章條文、教育訓練、日常提醒、佐證資料、違反後處理方式。
5. 違規事件不得只口頭處理，需留下提醒、輔導、改善追蹤或懲處紀錄。

## 建議優先產出文件

1. 特約契約違規記點與工作規章交叉比對表
2. 居服員服務紅線與違規風險說明
3. 居督日常提醒與輔導話術
4. 違規事件處理與改善追蹤表
5. 教育訓練簽到表與測驗題
